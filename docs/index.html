Интерактивная веб-платформа для презентации исследовательского проекта по физике. Проект получил высокую оценку экспертного сообщества и стал победителем престижных конференций.

Achievements

Проект успешно защищен и отмечен дипломами:

  • Республиканская конференция — один из лучших проектов
