# Matthew

**Physics Project Website** — это интерактивная веб-платформа, разработанная специально для презентации исследовательского проекта по физике. Проект получил высокую оценку экспертного сообщества и стал победителем престижных конференций.

---

### Achievements

Проект успешно защищен и отмечен дипломами на следующих площадках:
*   **Турнир юных физиков (ТЮФ)** — Призер 3 степени
*   **Республиканская конференция БГТУ "Первый шаг в науку"** — Диплом 3 степени

### Особенности
- **Интерактивные модели:** Визуализация физических процессов.
- **Точные расчеты:** Использование TypeScript для минимизации ошибок в формулах, и React для скорости website.
- **Высокая производительность:** Быстрая сборка и работа благодаря Vite.
